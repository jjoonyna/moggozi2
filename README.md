<h1>"랜선으로 즐기는 모꼬지!"</h1>
<Strong>모꼬지『표준국어대사전』</Strong><br>
: 놀이나 잔치 또는 그 밖의 일로 여러 사람이 모이는 일
<br><br><br>
<Strong>모꼬지</Strong>는 시간 및 거리의 제약으로 인해 자주 볼 수 없는 사람들을 위한<br> WebRTC를 사용하여 랜선으로 즐기는 취미, 소모임 사이트입니다.
